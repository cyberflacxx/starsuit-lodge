# Demo Guide

## Public Website Demo

- Open the homepage
- View branches
- View rooms
- Open gallery
- Open contact page

## Guest Booking Demo

- Open `/booking`
- Select Mutare or Chipinge
- Choose future dates
- Check availability
- Select a room
- Enter guest details
- Select a transport option
- Confirm the booking
- Go to the payment page
- Click `Pay Now`
- Click `Open EcoCash Dialer` on a phone
- Return and click `I Have Paid`
- Show the success page

## Admin Demo

- Login as super admin
- View dashboard
- Open bookings
- Open booking detail
- Verify payment as `PAID`
- Confirm booking
- Show the success page updated
- Open reports
- Show occupancy and revenue
- Open content management
- Edit branch or content

## Demo Note

The EcoCash USSD amount is fixed at `50` for the demo. The visible code remains `*153*1*1*0788064458*50#`, while the booking total still displays from the database.
